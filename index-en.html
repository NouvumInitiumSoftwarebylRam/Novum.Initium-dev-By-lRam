Logistic App - Warehouse Gate Management System (v1.5.0)

A modern logistics dashboard built as an IHK vocational training project during my interschip at the company HILA Automation for a possible customer, streamlining warehouse operations through automated driver check-ins and gate management.

Key Features:

  • Driver Self-Service Portal: Multi-language web interface for drivers to check-in using reference numbers, eliminating phone calls and paperwork
  • Real-Time Operator Dashboard: Live monitoring with drag-and-drop gate assignment, filtering, and status tracking
  • Automated Gate Assignment: Intelligent system that assigns gates automatically in 30%+ of cases
  • Robust Validation: Reduces reference errors by 50% through strict validation and clear error messages

Technical Stack:

  • Backend: Python FastAPI with SQLAlchemy, WebSockets, SQLite
  • Frontend: React 18 + TypeScript + Vite, Tailwind CSS, Zustand state management
  • Desktop: Electron wrapper for portable Windows deployment

Performance & Impact:

  • Reduces check-in time from 2-3 minutes to <1 minute
  • Streamlines warehouse operations with real-time updates
  • GDPR-compliant with minimal data retention
  • Offline-capable local deployment (no cloud dependency)
  • Successfully deployed at Faber-Castell facilities

Built during vocational training in Application Development, demonstrating full-stack development, UI/UX design, and production deployment skills.

Learning Management System

A comprehensive Learning Management System (LMS) built as a user-friendly full-stack web application. The platform provides complete functionality for managing online educational content and learning experiences.

Features: Secure user registration and authentication, course management for instructors with the ability to create courses and add lessons with educational materials, and an intuitive learning interface for students to view courses, complete lessons, and track their learning progress. The application supports integration of external resources such as PDFs and videos.

Technology Stack: Python Flask backend with SQLite database and React frontend. The project demonstrates core web development concepts including user authentication, database management, and RESTful API communication. The application runs locally with a Flask server on port 8000 and a React frontend on port 5173, facilitating straightforward setup and development.
