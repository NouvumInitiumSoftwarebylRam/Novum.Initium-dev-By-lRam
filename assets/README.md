# Assets Folder

## Required Files:

### 1. Logo
- **File name:** `logo.png`
- **Recommended size:** 200x200px or similar square dimensions
- **Format:** PNG (transparent background preferred)
- **Location:** `assets/logo.png`

### 2. Hero Background Image (NEW!)
- **File name:** `hero-bg.jpg.png` (current file)
- **Recommended size:** 1920x1080px or larger for best quality
- **Format:** JPG or PNG
- **Theme:** Futuristic/tech (robotic hand, circuits, etc.)
- **Location:** `assets/hero-bg.jpg.png`

The hero background image will appear behind your logo and text in the main hero section with a subtle overlay for readability.

### 3. Learning Management System Project Images
- **Folder:** `assets/lms/`
- **Required images:**
  - `dashboard.png` - Dashboard with welcome message and course overview
  - `explore-courses.png` - Explore Courses page with filters
  - `profile-settings.png` - Profile and Settings page
  - `grades.png` - Grades overview page
  - `user-profile.png` - User profile editing page
  - `package-json.png` - Project configuration screenshot
  - `python-auth.png` - Python authentication code screenshot
- **Recommended format:** PNG or JPG
- **Recommended size:** 1920x1080px or similar (screenshots)

## Bilingual Website Setup:
- `index.html` - German version (default)
- `index-en.html` - English version
- Language toggle buttons in the navigation
- Both versions share the same `styles.css` and `assets/` folder
